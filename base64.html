Base64 인코더 / 디코더

텍스트 ↔ Base64 양방향 변환. UTF-8 기반이라 한글·이모지도 깨지지 않습니다.

🔒 입력한 데이터는 브라우저에서만 처리되며 서버로 전송되지 않습니다.

광고 영역

사용법

위쪽 입력창에 텍스트를 넣고 인코딩 버튼을 누르면 아래 칸에 Base64 문자열이 생성됩니다. 반대로 아래 칸에 Base64 문자열을 붙여넣고 디코딩 버튼을 누르면 원문이 복원됩니다. 이 도구는 UTF-8 인코딩을 거쳐 변환하기 때문에 한글, 일본어, 이모지 등 멀티바이트 문자도 깨지지 않습니다. URL-Safe 옵션을 켜면 URL이나 파일명에 안전하게 쓸 수 있도록 +와 / 문자를 -와 _로 바꾼 변형(Base64URL)을 사용합니다.

이런 상황에서 쓰세요

  • JWT 토큰의 페이로드 부분을 디코딩해서 내용을 확인할 때
  • HTTP Basic 인증 헤더(Authorization: Basic ...)를 만들거나 해석할 때
  • 이메일 첨부나 설정 파일에 들어 있는 Base64 데이터를 원문으로 복원할 때
  • 바이너리 안전이 보장되지 않는 채널(JSON, XML 등)로 텍스트를 전달할 때
  • 쿠버네티스 Secret 값처럼 Base64로 저장된 설정 값을 읽거나 만들 때

Base64는 암호화가 아니라 단순 인코딩이므로 비밀번호 보호 용도로는 사용하면 안 됩니다. 이 도구의 모든 변환은 브라우저 내장 함수로 처리되며, 토큰이나 인증 정보 같은 민감한 값도 외부로 전송되지 않으니 안심하고 사용하세요.
