두 텍스트를 줄 단위로 비교해서 추가된 줄과 삭제된 줄을 색상으로 표시합니다.

🔒 비교할 텍스트는 브라우저에서만 처리되며 서버로 전송되지 않습니다.

광고 영역

사용법

왼쪽 칸에 변경 전 텍스트를, 오른쪽 칸에 변경 후 텍스트를 붙여넣고 비교하기 버튼을 누르세요. 두 텍스트를 줄 단위로 분석하여 삭제된 줄은 빨간색(− 기호), 추가된 줄은 초록색(+ 기호)으로 표시하고, 변화가 없는 줄은 그대로 보여줍니다. 비교 알고리즘은 git diff와 같은 원리인 최장 공통 부분 수열(LCS)을 사용하므로, 중간에 줄이 끼어들거나 빠져도 실제로 바뀐 부분만 정확하게 짚어냅니다. 상단에는 추가·삭제된 줄 수의 요약 통계도 함께 표시됩니다.

이런 상황에서 쓰세요

  • 설정 파일이 배포 전후로 어떻게 달라졌는지 비교할 때
  • 버전 관리에 올리지 않은 코드 조각 두 개의 차이를 빠르게 확인할 때
  • 계약서·공지문 등 문서의 수정 전후 버전에서 바뀐 문장을 찾을 때
  • API 응답이 환경(개발/운영)에 따라 어떻게 다른지 확인할 때
  • 번역문이나 리라이팅 결과에서 무엇이 바뀌었는지 검토할 때

git이 설치되지 않은 환경이나 저장소 밖의 파일을 비교할 때 특히 유용합니다. 계약서, 소스 코드처럼 민감한 내용도 브라우저 밖으로 나가지 않으므로 안심하고 비교하세요.
