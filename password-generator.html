비밀번호 생성기

원하는 길이와 문자 조합으로 예측 불가능한 랜덤 비밀번호를 만듭니다.

🔒 브라우저의 암호학적 난수(crypto API)로 생성되며 서버로 전송되지 않습니다.

강도:

광고 영역

사용법

슬라이더로 비밀번호 길이를 정하고, 포함할 문자 종류(대문자·소문자·숫자·특수문자)를 체크박스로 선택한 뒤 생성 버튼을 누르면 조건에 맞는 랜덤 비밀번호가 즉시 만들어집니다. 선택한 문자 종류가 최소 하나씩은 결과에 포함되도록 만들어지며, 나머지 자리는 무작위로 채워진 뒤 순서를 다시 섞습니다. 아래에는 길이와 문자 다양성을 기준으로 약함/보통/강함 강도가 표시되므로, 강도가 부족하면 길이를 늘리거나 문자 종류를 추가하세요. 복사 버튼으로 클립보드에 바로 담아 비밀번호 관리자나 가입 양식에 붙여넣을 수 있습니다. 난수 생성에는 crypto.getRandomValues를 사용해 UUID 생성기와 동일하게 예측이 어려운 안전한 방식으로 만듭니다.

이런 상황에서 쓰세요

  • 새로 가입하는 서비스의 계정 비밀번호를 안전하게 만들 때
  • 임시 발급용 초기 비밀번호(신규 직원 계정 등)를 여러 개 만들어야 할 때
  • Wi-Fi, 공유기 관리자 비밀번호처럼 자주 안 바꾸는 비밀번호를 강하게 설정할 때
  • 테스트 계정이나 API 키의 시크릿 값을 임시로 생성할 때
  • 회사 비밀번호 정책(길이·특수문자 포함 등)에 맞는 값을 빠르게 만들 때

생성된 비밀번호는 화면에만 표시되고 서버로 전송되거나 저장되지 않으므로, 새로고침하면 같은 값을 다시 볼 수 없습니다. 중요한 계정에 사용할 비밀번호는 생성 직후 비밀번호 관리자에 즉시 저장해 두는 것을 권장합니다. 비밀번호를 더 안전하게 관리하는 방법이 궁금하다면 안전한 비밀번호를 만드는 5가지 원칙을 참고하세요.

자주 묻는 질문

Q. 생성된 비밀번호는 안전한가요?

A. 브라우저의 암호학적 난수(crypto.getRandomValues)로 만들어져 예측이 매우 어렵습니다.

Q. 강도가 "약함"으로 나오면 어떻게 해야 하나요?

A. 길이를 12자 이상으로 늘리고 대문자·숫자·특수문자 등 문자 종류를 더 선택하세요.

Q. 특수문자를 지원하지 않는 사이트는 어떻게 하나요?

A. 특수문자 체크박스를 해제하고 길이를 더 늘려 강도를 보완하세요.
