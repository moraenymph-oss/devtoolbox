Cron 표현식 생성기

각 필드를 선택하면 크론 표현식이 완성되고, 의미가 한글로 표시됩니다. (표준 5필드: 분 시 일 월 요일)

🔒 모든 처리는 브라우저에서만 이루어지며 서버로 전송되지 않습니다.

* * * * *

필드 문법 참고

기호의미예시
*모든 값매분, 매시…
숫자특정 값0 = 정각/자정
a-b범위1-5 = 월~금
a,b,c목록0,30 = 0분과 30분
*/nn 간격마다*/10 = 10분마다
광고 영역

사용법

다섯 개의 입력칸(분·시·일·월·요일)에 값을 넣으면 아래에 완성된 크론 표현식과 한글 해석이 실시간으로 표시됩니다. 직접 입력하는 대신 프리셋 버튼을 눌러 "매일 자정", "평일 오전 9시" 같은 자주 쓰는 패턴에서 시작해 수정할 수도 있습니다. 각 필드에는 *(모든 값), 1-5(범위), 0,30(목록), */10(간격) 문법을 조합해서 쓸 수 있으며, 요일은 0(일요일)부터 6(토요일)까지입니다. 완성된 표현식은 표현식 복사 버튼으로 바로 복사해 crontab에 붙여넣으세요.

이런 상황에서 쓰세요

  • 리눅스 crontab에 백업 스크립트 스케줄을 등록할 때
  • Jenkins, GitHub Actions(schedule), Airflow 등 CI/배치 도구의 실행 주기를 설정할 때
  • 스프링 @Scheduled나 node-cron의 표현식을 만들 때 (초 필드가 있는 6필드 문법은 맨 앞에 초를 추가하세요)
  • 운영 중인 배치의 크론 표현식이 정확히 언제 실행되는 건지 해석해야 할 때
  • "평일 점심시간만 제외하고 실행" 같은 복잡한 조건을 검증할 때

크론 표현식은 시스템마다 시간대(타임존) 기준이 다르다는 점에 주의하세요. 서버가 UTC로 동작한다면 한국 시간 오전 9시는 0 0 * * *(UTC 0시)로 지정해야 합니다. 이 도구의 모든 계산은 브라우저에서만 이루어집니다.
