DevToolbox 소개

왜 만들었나요?

개발을 하다 보면 JSON 정리, Base64 디코딩, 타임스탬프 변환처럼 아주 작은 작업이 수시로 필요합니다. 그때마다 검색해서 낯선 사이트에 회사 데이터를 붙여넣는 것은 번거롭고, 보안 측면에서도 찜찜합니다. DevToolbox는 이런 반복 작업을 한곳에서, 데이터 유출 걱정 없이 처리할 수 있도록 만든 개인 프로젝트입니다.

운영 원칙

  • 100% 클라이언트 처리 — 모든 도구는 순수 HTML/CSS/JavaScript로 작성되어 브라우저 안에서만 실행됩니다. 입력한 데이터는 서버로 전송되지 않습니다.
  • 회원가입 없음 — 어떤 도구도 로그인이나 개인정보 입력을 요구하지 않습니다.
  • 무료 — 모든 기능은 무료이며, 사이트 운영을 위해 광고가 게재될 수 있습니다.
  • 가볍고 빠르게 — 프레임워크 없이 만든 정적 페이지라 어떤 환경에서도 빠르게 열립니다.

제공 중인 도구

JSON 포매터/검증기, 글자수·바이트 세기, Base64 인코더/디코더, URL 인코더/디코더, 정규식 테스터, Cron 표현식 생성기, 타임스탬프 변환기, UUID 생성기, HEX↔RGB 색상 변환기, 텍스트 Diff 비교기 등 총 10가지 도구를 제공하고 있으며, 앞으로도 실무에서 자주 쓰이는 도구를 계속 추가할 예정입니다.
