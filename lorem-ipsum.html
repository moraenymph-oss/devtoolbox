사용법

언어(라틴어 Lorem Ipsum 또는 한국어)와 문단 수를 지정한 뒤 생성 버튼을 누르면, 문단마다 여러 개의 무작위 문장으로 구성된 더미 텍스트가 만들어집니다. 실제 의미가 없는 자리채움용 텍스트이므로 디자인 시안에서 폰트 크기, 줄 간격, 레이아웃의 균형만 확인할 때 적합합니다. 복사 버튼으로 클립보드에 담아 디자인 툴이나 퍼블리싱 코드에 바로 붙여넣을 수 있고, 생성된 텍스트의 정확한 글자 수를 확인하고 싶다면 글자수·바이트 세기 도구와 함께 사용하면 편리합니다.

이런 상황에서 쓰세요

  • 웹·앱 디자인 시안에서 실제 카피 대신 채워 넣을 임시 텍스트가 필요할 때
  • 퍼블리싱 작업 중 레이아웃이 다양한 텍스트 분량에서도 깨지지 않는지 테스트할 때
  • 블로그·게시판 기능을 개발하면서 테스트용 게시글 데이터를 빠르게 채울 때
  • 프레젠테이션이나 목업에서 텍스트 영역의 느낌만 먼저 보여주고 싶을 때
  • 폰트·타이포그래피 조합을 테스트할 때 실제 문구에 시선이 뺏기지 않게 하고 싶을 때

Lorem Ipsum은 실제 라틴어 문장이 아니라 인쇄·출판업계에서 오래전부터 자리채움용으로 써온 관용적인 텍스트이며, 한국어 더미 텍스트 옵션은 의미 없는 단어를 무작위로 조합해 비슷한 용도로 쓸 수 있게 만든 것입니다. 생성은 전적으로 브라우저에서 이루어지며 별도 데이터 전송은 없습니다.

자주 묻는 질문

Q. 한국어 더미 텍스트는 실제 문장인가요?

A. 아니요. 의미 없는 단어를 무작위로 조합한 자리채움용 텍스트입니다.

Q. 문단당 문장 수를 조절할 수 있나요?

A. 현재는 자동으로 4~8문장 사이에서 무작위로 생성됩니다.

A. 고대 라틴 문헌 일부를 변형해 만들어진 관용적인 자리채움 텍스트이기 때문입니다.
