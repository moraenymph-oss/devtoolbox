타임스탬프 변환기

유닉스 타임스탬프 ↔ 날짜/시간을 서로 변환합니다. 한국 시간(KST)과 UTC를 함께 표시합니다.

🔒 모든 계산은 브라우저에서만 이루어지며 서버로 전송되지 않습니다.

사용법

맨 위에는 현재 시각의 유닉스 타임스탬프가 1초 단위로 갱신되어 표시됩니다. 타임스탬프 → 날짜 칸에 숫자를 입력하면 10자리(초)인지 13자리(밀리초)인지를 자동으로 판별하여 한국 시간(KST, UTC+9)과 협정세계시(UTC), 그리고 ISO 8601 형식으로 변환해 줍니다. 반대로 날짜 → 타임스탬프에서는 달력에서 날짜와 시간을 고르면 초·밀리초 단위 타임스탬프를 모두 보여줍니다. 유닉스 타임스탬프는 1970년 1월 1일 0시(UTC)부터 흐른 시간을 뜻하며, 전 세계 어디서나 같은 값을 갖습니다.

이런 상황에서 쓰세요

  • 서버 로그나 DB에 저장된 created_at 타임스탬프가 실제 몇 월 며칠인지 확인할 때
  • JWT 토큰의 만료 시각(exp) 값을 사람이 읽을 수 있는 시간으로 바꿀 때
  • API 요청 파라미터에 넣을 특정 날짜의 타임스탬프가 필요할 때
  • UTC 기준 로그와 한국 시간 사이의 9시간 차이를 헷갈리지 않고 확인할 때
  • 캐시 만료 시각, 예약 발송 시각 등을 계산할 때

변환은 자바스크립트 Date 객체로 브라우저 안에서만 처리됩니다. 초 단위와 밀리초 단위를 혼동하면 1970년이나 먼 미래로 계산되는 흔한 실수를 자동 판별 기능이 막아줍니다.
